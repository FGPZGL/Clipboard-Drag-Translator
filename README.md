The program's UI supports only Korean.

This program was developed using Python. ![python](https://img.shields.io/badge/Python-14354C?style=for-the-badge&logo=python&logoColor=white)
This program was developed for the Windows operating system. ![windows](https://img.shields.io/badge/Windows-0078D6?style=for-the-badge&logo=windows&logoColor=white)
This program uses Google Translator. ![google](https://img.shields.io/badge/Google-4285F4?logo=google&logoColor=fff&style=for-the-badge)
Translation speed may vary depending on the user's computer specifications, internet connection, Google server status, and other factors.
It automatically detects the source text and translates it into your desired language.
You can select the language to translate into in the settings.
You can access quick settings by right-clicking.

This program supports the following languages.
```txt
┌─ Korean
├─ English
├─ private
├─ Japanese
├─ Chinese
├─ French
└─ German
```
